# خطا های رایج

# kInstallDeviceOpenError
اگر این خطا را در زمان فلش یک رام اندرویدی میبینید، دلیل آن این است که گوشی شما روی Slot A قرار دارد و آن رام نمیتواند به همان Slot فلش شود، به بوتلودر بروید و دستور زیر را وارد کنید:
```bash
fastboot set_active b
```
سپس دوباره به ریکاوری بروید و رام مورد نظر را فلش بزنید.
```bash
fastboot reboot recovery
```
